Introduction¶

Dataset Description¶

Data info: This dataset collects information from 100k medical appointments in Brazil and is focused on the question of whether or not patients show up for their appointment. A number of characteristics about the patient are included in each row.

Reason for Choosing this Dataset¶

i like to analyze medical data and Disaster-specific data, especially air disasters because its related to serve humanity and medical stuff to get a decession about their data

Column Names:¶

  • PatientId: The patient id
  • AppointmentID: The appointment id of the patient
  • Gender: gender of each patient male or female
  • ScheduledDay: the day which the patient set up their appointment
  • AppointmentDay: appointment day for the patient
  • Age: age of the patient
  • Neighbourhood: location of the hospital
  • Scholarship: whether the patient is enrolled in Brasilian program or not (1 = enrolled, 0 = notenrolled)
  • Hipertension: determine the patient suffers from a hipertension or not (1 = suffer , 0 = not suffer)
  • Diabetes: determine the patiendt suffers from a diabetes or not (1 = suffer , 0 = not suffer)
  • Alcoholism : determine the patient blood contains of Alcoholism or not (1 = contains , 0 = not contains)
  • Handcap: determines the patient disability degree --> (0, 1, 2, 3, 4)

    • In Brazil, disability is measured by facts and age-based data sport participation education information impacts on everyday life and available support services/issues of access for more info: click here

  • SMS_received: reminder message about appointment day for the patient (1 = received , 0 = not received)
  • No-show: determines the patient attended or not in the appointment day (Yes = not attended , No = attended)

# fuction for hist charts its take 6 arguments (dataframe , coloumn name , title, xlabels title and ylabel title ) where:
# dataframe is the data frame that have our file with its data, coloumn name is the field inside this dataframe
# title is the title of the hist chart, xlabel is the title for x-axis label and ylabel is the y-axis title, case argument
#takes 0 or 1 if 0 this means first case that plot simple histogram on one column such as period
#case 2 if the value 1 --> work on the effect of coloumn to another such as age with no-show, period with no-show , etc
# the function returns a histogram chart for our data frame coloum based on the choosen case
def draw_hist_chart(df, col, title, xlabel, ylabel, case):
    fig = plt.figure(figsize = (7,5))    # controls figure size
    ax = fig.gca() # Get the current Axes instance on the current figure matching the given keyword args, or create one.
    plt.title(title, fontsize=22);
    plt.xlabel(xlabel, fontsize=18) # font size of x label
    plt.ylabel(ylabel, fontsize=18);# font size of y label
    plt.xticks(fontsize=14) # font size of x ticks
    plt.yticks(fontsize=14) # font size of y ticks
    if case==0:
        df[col].hist(ax=ax)
    elif case==1:
        show_yes = df.No_show=='Yes'     # get from attribute No_show Yes values only
        show_no = df.No_show=='No'       # get from attribute No_show No values only
        df[col][show_yes].hist(alpha=0.5, label='Not Attended', ax=ax)    # not if this not working look at HTML file 
        df[col][show_no].hist(alpha=0.5, label='Attended', ax=ax)         #it works well at my operating system
        plt.legend()

In general:

  • variables with correlation = 1 , but this is because between the variable and itself
  • variables with correlation between 0 and (+/-) 0.5 are weak correlation
  • variables with correlation between (+/-) 0.5 and (+/-) 0.7 are intermediate correlation
  • variables with correlation between (+/-) 0.7 and (+/-) 1 are strong correlation
  • variables with correlation = 0 refers to those variables are not correlated
where (+)--> positive correlation and (-) --> is a negative correlation

for example :
The relation between HiperTension and age is intermediate positive correlation (0.504586)
    this means if age increase then the propability of hipertension for this patient is increase too
The relation between Period and No_show is weak negative corelation (-0.186256)
   this means if the period increases then the propability of the patient attendence decreases
